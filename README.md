# Pusula KPSS - Landing Page

Welcome! This repository contains the source code for the promotional landing page of **[Pusula YKS / Pusula KPSS]**.

### 🚀 Note on the Core Application
Please note that the main application is a comprehensive AI-driven digital coaching platform developed as a closed-source (private) project. The core full-stack architecture is built with:

* **Mobile/Frontend:** React Native & Expo
* **Backend & Database:** Node.js, Supabase, Prisma
* **Automation & Payments:** n8n & Shopier API
* **Deployment & CI/CD:** Vercel & Render

Due to the commercial nature of the product, the core source code is kept private.
